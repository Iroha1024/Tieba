## 使用详情

```bash
git clone https://github.com/Iroha1024/tieba.git

npm install
修改数据库连接(server/db.js)
node server/server.js
npm run dev

```

> 在使用打包时，请注意自己的编译环境，详情见
> 
> https://simulatedgreg.gitbooks.io/electron-vue/content/cn/getting_started.html

```bas
npm run build:win32
```
